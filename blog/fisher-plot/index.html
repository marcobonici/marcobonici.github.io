    FisherPlot.jl: plotting your Fisher posterior contours

    FisherPlot.jl is my first registered Julia package. It can be used to easily plot Fisher matrix contours. In the remainder of this post, I'll give a (short) introduction to Fisher matrices, before showing how actually use this package. Cool, isn't it?

    Fisher Matrix

    The Fisher Matrix is a tool that can be used to make forecasts, i.e. to predict the sensitivity of an experiment to a set of parameters. Given the likelihood, the expression of the Fisher matrix is given by[1]

    Fij≡−⟨∂2log⁡L∂θi∂θj⟩; \boldsymbol{F}_{i j} \equiv-\left\langle\frac{\partial^{2} \log L}{\partial \theta_{i} \partial \theta_{j}}\right\rangle; Fij​≡−⟨∂θi​∂θj​∂2logL​⟩;

    i.e. the Fisher matrix is given by (minus) the Hessian of the log likelihoodlog⁡L\log LlogL.
